# Spec2Vec
This repository has been jointly developed by <b>PSU-GeoFluids-Lab and <b>URseismology Lab

## Short Description
Spec2VEC is an intelligent framework designed to extract objective, interpretable feature embeddings from Time-Frequency Representations (TFRs). The tool computes a suite of information-theoretic, statistical and spatial metrics to generate a unique "spectral fingerprint."

Developed specifically for geophysics and planetary exploration (but not limited to this domain), Spec2VEC addresses the "cold-start" problem where labeled training data is unavailable. It provides a transparent, physics-aligned (does not learn physics but algortihm signatures) alternative to "black-box" deep learning models, ensuring that signal characterization remains consistent and reproducible.
